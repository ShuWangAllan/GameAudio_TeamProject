This is for Game Audio  Team Project.  

The option our group choose:  
Sonify a given pre-made game.  

The software requires:  
- Unity
- FMode Studio
- Github Desktop
- VS or VS Code with C# kit

required to provide a professional audio implementation for a small Unity game using FMOD Studio. This includes, but is not limited to:
• All area sounds
• All source loops
• All character sounds
• All weapon sounds
• All interactable object sounds
• All menu sounds
